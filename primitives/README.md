# Cortex Primitives

Low-level reusable UI building blocks live here.

- `layout`: structural layout primitives — grid, stack, inline, split
- `actions`: buttons and direct actions — button, icon-button, split-button, button-group
- `inputs`: field-like controls — text/number/date fields, select, checkbox, radio, switch, slider, file upload
- `navigation`: tabs, breadcrumbs, and pagination
- `overlay`: dialogs, popovers, menus, tooltips, and the lightbox
- `feedback`: status and progress — alerts, banners, toasts, loaders, progress bars, validation messages
- `display`: content and presentation — cards, tags, badges, avatars, metrics, markdown, code blocks
- `data`: data-bearing views — tables, lists, trees, charts, query fields
- `media`: image and icon primitives
